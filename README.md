## Resumo do Projeto

Este projeto implementa um sistema de gerenciamento de heróis utilizando os conceitos de Programação Orientada a Objetos (POO) em JavaScript. A aplicação permite criar personagens com diferentes atributos e habilidades, simulando um ambiente de jogo de RPG simplificado.

## Características

O sistema possui uma classe heroi que encapsula as propriedades e comportamentos dos personagens:

- **Atributos**: nome, classe, idade, tipo e ataque
- **Métodos**: 
  - escrever() - exibe informações completas do herói
  - atacar() - mostra o tipo de ataque utilizado
  - tipoAtaque() - identifica a especialização do herói baseada no tipo de ataque

## Funções

O projeto demonstra conceitos fundamentais de POO como:

- Criação de classes e instanciação de objetos
- Encapsulamento de dados e comportamentos
- Métodos para interação com os objetos
- Lógica condicional para diferentes tipos de ataques

## Exemplos de Heróis

O código inclui dois personagens pré-configurados:

- **Saitama**: Guerreiro classe SS especializado em artes marciais
- **Frieri**: Maga classe SS com domínio de magia ancestral

Cada herói possui características únicas que determinam seu estilo de combate e habilidades especiais no sistema.
